## About
This repository contains the code for the __Mendel__ module of the __Fast Data Collection__ project. This module is a database microsservice that is going to be used by other services of the project. This is a project done for the [LPROT](http://www.pea.usp.br/pesquisa/linha-de-pesquisa/lprot-laboratorio-de-pesquisa-em-automacao-e-protecao-de-sistemas-eletricos/) laboratory of the [Polytechnic School of the University of SÃ£o Paulo](https://www.poli.usp.br/).

# Features
* Receive and forward data via a Kafka Queues
* Allow for another service to query data for given time ranges
